# 🚧 Road Accident Insights Dashboard

## 🚀 Overview

This dashboard analyzes road accident data to identify patterns, causes, and high-risk areas.

---

## 📊 Features

* Accident trends over time
* Severity analysis
* Location-based insights
* Vehicle involvement
